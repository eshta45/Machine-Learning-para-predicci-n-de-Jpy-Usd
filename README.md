# Machine-Learning-para-predicci-n-de-Jpy-Usd

**El problema que se intenta resolver.**
Los precios de las divisas pueden ser muy impredecibles, este sistema busca fácilitar la predicción de los mismos.

**El conjunto de datos utilizado.**
data historica de este cambio de divisa extraido de https://www.investing.com/currencies/usd-jpy-historical-data.

**El enfoque de modelado implementado.**
El modelo consta del entrenamiento de modelos de clasificación, para simplemente identificar si el precio subirá o bajará y de modelos de regresión que buscan ayudarnos a predecir aproximadamente el precio que presentará en una fecha dada.

**La estructura del repositorio.**
El repositorio simplemente consta de un cuaderno ejecutable con el código y sus respectivos comentarios, el reporte del proyecto, un archivo de texto con las librerias de python necesarias para ejecutar el programa y la base de datos utilizada.

**Instrucciones para ejecutar el proyecto y reproducir los resultados.**
Instalar las librerias de Requirements.txt previo a correr el código en Proyecto.ipynb.
